# Claude Token Usage

[![Claude Code plugin](https://img.shields.io/badge/Claude_Code-plugin-D97757)](https://code.claude.com/docs/en/plugins)
[![License: AGPL v3](https://img.shields.io/badge/license-AGPL_v3-blue.svg)](LICENSE)
[![Dependencies](https://img.shields.io/badge/dependencies-bash%20%2B%20jq-4EAA25)](#requirements)

See the tokens used by each Claude Code prompt-to-response turn. No API calls, telemetry, or estimation: the plugin reads Claude Code's local JSONL transcript after the response stops.

## Showcase

```text
Claude's response ends here.
└─ Stop says: 15:57:37 | turn total: 5,685 tokens (fresh input 5, cache read 5,480, cache new 100, output 100)
```

The total covers every distinct model call Claude made after your latest prompt, including tool-use loops.

## Install

### From GitHub

```bash
claude plugin marketplace add wilmtang/claude-token-usage
claude plugin install claude-token-usage@claude-token-usage
```

Run `/reload-plugins` in Claude Code, or start a new session.

### From a local clone

```bash
git clone https://github.com/wilmtang/claude-token-usage.git
claude plugin marketplace add ./claude-token-usage
claude plugin install claude-token-usage@claude-token-usage
```

Remove any older `stop-token-usage.sh` entry from `~/.claude/settings.json` before installing, otherwise both hooks will print.

## Requirements

- Claude Code with plugin support
- Bash
- [`jq`](https://jqlang.org/download/)

On macOS: `brew install jq`. On Debian or Ubuntu: `sudo apt install jq`.

## How it works

Claude Code passes the `Stop` hook a local transcript path. The hook:

1. Finds the latest real user prompt, ignoring tool-result and metadata messages.
2. Reads usage from each distinct assistant API request after that prompt.
3. Deduplicates repeated transcript records by request or message ID.
4. Prints one comma-formatted summary through the hook's `systemMessage` field.

| Field | Meaning |
| --- | --- |
| `fresh input` | Input tokens not served from prompt cache |
| `cache read` | Input tokens reused from prompt cache |
| `cache new` | Input tokens written to prompt cache |
| `output` | Tokens generated by the model |
| `turn total` | Sum of the four fields across all model calls in the turn |

`turn total` is processed token volume, not context-window size, subscription usage, or a dollar-cost estimate. It can be large when one turn makes many model calls or repeatedly reads a large cached context.

## Verify

```bash
./tests/test-hook.sh

```

## Privacy

Everything runs locally. The hook reads only the transcript path Claude Code provides and makes no network requests.
